## Purpose

This ZIP file includes all necessary source code, resource files, and configuration scripts (`build.gradle`, etc.) required to compile and run the final application successfully in a separate Android Studio environment. It fulfills all requirements, including the implemented **Triple-Mode Quiz Engine**, **Advanced High Score Persistence**, **Custom Game Logic**, and **Full Resource Integration** (78 municipalities, sounds, colors, strings).
